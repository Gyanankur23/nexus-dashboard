### Added
- Initial release of Nexus Dashboard
- Admin panel with user management
- Project management with Kanban board
- Task tracking with priority levels
- Real-time dashboard statistics
- JWT authentication with SHA-256 password hashing
- Dark high-contrast UI design
- Responsive layout for all screen sizes
- Vue.js 3 frontend with Pinia state management

- RESTful API endpoints
- Comprehensive documentation
